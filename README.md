## ğŸš€ Getting Started <a name="-getting-started"></a>

First, install the dependencies. We recommend using `npm` for this project.

```bash
npm install
```

Then, run the development server:

```bash
npm run sever
# or
yarn server
# or
pnpm server
# or
bun server
```

Then, run the development client:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) and [http://localhost:5173](http://localhost:5173) with your browser to see the result.
